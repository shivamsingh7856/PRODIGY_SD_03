# Contact Management System - Java

## Description
The Contact Management System is a Java console application that allows users to store and manage contact information. Users can add new contacts, view the contact list, edit existing contacts, and delete contacts. The program stores contact data in a file to ensure persistent storage, so the data is saved even after the program is closed.

## Features
- Add new contact
- View contact list
- Edit contact
- Delete contact
- File storage (persistent data)
- User-driven menu system

## Technologies Used
- Java
- ArrayList
- File Handling
- OOps Concepts
